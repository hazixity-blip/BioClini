Code systems explained

Each system serves a different purpose. Researchers often need several at once.

ICD-10-CM

International Classification of Diseases

The global standard for billing, EHR records, and epidemiology. Used in claims databases and hospital records. Maintained by WHO/CDC.

icd.who.int ↗
RxNorm

RxNorm (RXCUI)

NLM's normalized drug naming system. The RXCUI is the canonical ID for a drug concept — used to link across pharmacy, EHR, and claims systems.

nlm.nih.gov ↗
NDC

National Drug Code

FDA's product identifier for a specific drug product, dosage, and manufacturer. Required for pharmacy claims. 10-11 digit code.

fda.gov ↗
SNOMED CT

SNOMED Clinical Terms

The most comprehensive clinical terminology. Used in interoperability, HL7 FHIR, and clinical decision support. ~350,000 concepts.

snomed.org ↗
MeSH

Medical Subject Headings

NLM's controlled vocabulary for indexing biomedical literature. Used to search PubMed and index research papers. Essential for literature review.

nlm.nih.gov ↗
ATC

WHO ATC Code

Anatomical Therapeutic Chemical classification. Hierarchical 7-character code that classifies drugs by body system and therapeutic action.

whocc.no ↗

Built for the gap between two worlds

Biotech researchers and clinical data teams speak different languages. A researcher knows "metformin" — the EHR query needs RXCUI:6809, the claims query needs NDC:0093-1074, the paper search needs MeSH:D008687.

MedCode Translator closes that gap. It calls free public APIs from NLM, FDA, and SNOMED — no sign-up, no cost, no data sent to any server.
